ShareYourSounds

버즈 하나로 여러 기기의 소리를 동시에.
노트북에서 앱을 실행하면 QR코드가 뜨고,
폰에서 찍으면 바로 오디오가 전달됩니다.

최신 버전 다운로드

Windows · Linux (PulseAudio) · AppImage / .deb / .msi

사용 방법

1
앱 실행 노트북 또는 데스크톱에서 ShareYourSounds를 실행합니다.
2
Open Host & Wait 버튼을 누르면 로컬 서버가 열리고 QR코드와 URL이 표시됩니다.
3
폰에서 QR 스캔 같은 와이파이에 연결된 폰으로 QR을 찍으면 오디오가 실시간 전달됩니다.

방화벽 설정

모바일 기기에서 접속하려면 TCP 6767 포트를 열어야 합니다.

# Linux · ufw
sudo ufw allow 6767/tcp

# Linux · firewalld
sudo firewall-cmd --add-port=6767/tcp --permanent
sudo firewall-cmd --reload
